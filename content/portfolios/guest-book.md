### Key Features

The dynamic web-based Guest Book application provides administrators with tools to manage guest lists and entries efficiently. The key features include:

### Admin Features

- Allows administrators to access the system securely.
- Enables administrators to add, edit, and delete guest book entries.
- Allows administrators to manage the guest list, including editing and deleting guests.
- Generates a report of all guests who have recorded their presence in the guest book.
- Enables administrators to manage other admin accounts, including adding, editing, and deleting admin accounts.
- Allows administrators to update their profile information.
- Securely logs out administrators from the system.

This application streamlines the guest management process, making it easy for administrators to oversee guest lists and entries effectively.
