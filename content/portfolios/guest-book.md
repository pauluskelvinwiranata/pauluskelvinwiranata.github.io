## Buku Tamu

Aplikasi **Buku Tamu** berbasis web yang dinamis menyediakan alat bagi administrator untuk mengelola daftar dan entri tamu secara efisien. Fitur utama meliputi:

### Fitur Admin

- **Akses Sistem**: Memungkinkan administrator mengakses sistem.
- **Manajemen Entri Buku Tamu**: Memungkinkan administrator menambah, mengedit, dan menghapus entri buku tamu.
- **Manajemen Daftar Tamu**: Memungkinkan administrator mengelola daftar tamu, termasuk mengedit dan menghapus tamu.
- **Laporan Tamu**: Menghasilkan laporan semua tamu dalam periode tertentu.
- **Manajemen Akun Admin**: Memungkinkan administrator mengelola akun admin lainnya, termasuk menambah, mengedit, dan menghapus akun admin.
- **Pembaruan Profil**: Memungkinkan administrator memperbarui informasi profil mereka.
- **Logout**: Keluar dari sistem.

### Teknologi

Aplikasi ini dibangun dengan menggunakan teknologi utama sebagai berikut:

- **Bootstrap**
- **PHP**
- **MySQL**

Aplikasi ini mempermudah proses manajemen tamu, sehingga administrator dapat mengawasi daftar dan entri tamu dengan lebih efektif.
