### 🌟 Key Features 🌟

The dynamic web-based Guest Book application provides administrators with tools to manage guest lists and entries efficiently. The key features include:

### 🔧 Admin Features

| Feature                | Description                                                                                     |
|------------------------|-------------------------------------------------------------------------------------------------|
| **Login**              | Allows administrators to access the system securely.                                             |
| **Guest Book Management** | Enables administrators to add, edit, and delete guest book entries.                             |
| **Guest List Management** | Allows administrators to manage the guest list, including editing, and deleting guests. |
| **Guest Report** | Provides a report of all guests who have signed the guest book.  |                              |
| **Admin Management**   | Enables administrators to manage other admin accounts, including adding, editing, and deleting admin accounts. |
| **Profile Updates**     | Allows administrators to update their profile information.                                       |
| **Log Out**            | Securely logs out administrators from the system.                                                |

This application streamlines the guest management process, making it easy for administrators to oversee guest lists and entries effectively.
