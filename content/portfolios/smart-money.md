### 🌟 Key Features 🌟

"Smart Money" is a web-based application designed to help users manage their personal finances, including income and expenses. The application offers the following features:

### 👤 User Features

| Feature                | Description                                                                                      |
|------------------------|--------------------------------------------------------------------------------------------------|
| **Register**           | Allows new users to register and create a new account.                                           |
| **Login**              | Allows users to log into the system with secure credentials.                                     |
| **CRUD Categories**    | Manage financial categories, including adding, modifying, deleting, and viewing categories.      |
| **CRUD Transactions**  | Manage financial transactions, including adding, modifying, deleting, and viewing transactions.  |
| **Update Profile**     | Allows users to update their profile information.                                                |
| **Financial Report Summary** | Provides financial reports summarizing income and expenses over a specified period.          |
| **Log Out**            | Securely log out of the system after completing activities.                                       |

This application is designed to offer ease and flexibility in managing personal finances, helping users track and organize their income and expenses more effectively.
