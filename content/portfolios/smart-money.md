### Key Features

"Smart Money" is a web-based application designed to help users manage their personal finances, including income and expenses. The application offers the following features:

### User Features

- Allows new users to register and create a new account.
- Allows users to log into the system with secure credentials.
- Manage financial categories, including adding, modifying, deleting, and viewing categories.
- Manage financial transactions, including adding, modifying, deleting, and viewing transactions.
- Allows users to update their profile information.
- Provides financial reports summarizing income and expenses over a specified period.
- Securely logs out users from the system.

This application is designed to offer ease and flexibility in managing personal finances, helping users track and organize their income and expenses more effectively.
